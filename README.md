# GREENBLOCK
GreenBlock es una plataforma educativa que transforma el reciclaje escolar en un modelo gamificado Recycle-to-Earn. Fusiona Inteligencia Artificial para clasificar residuos, Blockchain para asegurar la trazabilidad de datos y otorgar recompensas, y Biotecnología con hongos biodegradadores para destruir el plástico en BioLabs.
